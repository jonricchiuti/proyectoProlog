ESTADO ACTUAL DEL PROYECTO.

Este proyecto cumple con todas las especificaciones pero con ciertas considereciones. Para generar las sopas de letras se decidio
utilizar un backtracking puro, por esta razon, para sopas de letras muy grandes (en cuando a dimensiones y/o a alfabeto) el 
procesamiento de las mismas es muy lento. 

En ciertos casos se arrojan muchas sopas de letras iguales. 

Cuando el programa le pide al usuario si quiere otra sopa de letras, y se le suministra algo diferente de "no." esto es tomado como un 
"mas.".
